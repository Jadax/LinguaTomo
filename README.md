# LinguaTomo

A warm, local-first Japanese learning companion for Web, Android and iOS.

Created by Tushant Sharma. An Astraiva product.

## Release 1.2.0

- 828 searchable Japanese grammar lessons spanning N5 to N1 study references
- FSRS Memory Garden for both situational phrases and grammar patterns
- 30 Living Postcards, 24 seasonal story side quests and 28 achievements
- practical Can-Do route with separate JLPT, CEFR and ILR reference points
- kana, kanji, touch writing, paper handwriting OCR and warm Japanese playback
- offline-first Hive progress with optional Supabase account sync
- responsive Flutter interface for Web, Android and iOS

## Content provenance

The included Hanabira grammar corpus is CC BY-SA 3.0. Full attribution,
revision details and exclusions are documented in
[`THIRD_PARTY_CONTENT.md`](THIRD_PARTY_CONTENT.md) and displayed in the app.
JLPT labels are study references, not official content lists or certificates.
