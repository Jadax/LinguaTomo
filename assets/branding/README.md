# Brand assets

`leo-cheer.png` and `leo-nest-fireplace.png` are project-owned companion
artwork created for LinguaTomo. Leo is represented as an illustrated Norwegian
Forest Cat, not as a user avatar.

`leo-face-icon.png` is the close-cropped Leo identity used for the favicon,
passport and install icons. `nest-spring-pixel.png` and
`nest-winter-pixel.png` extend the same restrained, cosy pixel-art environment
system. They were generated specifically for LinguaTomo with OpenAI's built-in
image generation tool on 22 July 2026, then reviewed and resized locally. The
original full-resolution outputs are retained in the Codex generated-images
workspace for this build.

All Nest backgrounds reserve a quiet central or lower-floor display area. The
application places a maximum of eight earned objects there, keeping the room
legible on phones and preventing a collection from turning into visual clutter.

`astraiva.png` is retained as a private brand-source reference in the
repository. It is deliberately not declared as a Flutter asset and must not be
placed in learner-facing screens. Product ownership is expressed in metadata,
legal copy and source documentation instead.
